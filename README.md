# NOVA TALKING AI ASSISTANT

NOVA is a Python-based voice assistant that can listen, speak, fetch news headlines, crack jokes, and perform basic automation tasks.

## Features
- Voice recognition using microphone
- Text-to-speech responses
- Latest news headlines via Google News RSS
- Jokes for fun interactions
- Modular code structure

## Tech Stack
- Python
- speech_recognition
- pyttsx3
- feedparser
- OpenAI API
