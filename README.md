<h1>NOVA TALKING AI ASSISTANT</h1>
<br>

NOVA is a Python-based voice-controlled AI assistant designed to interact naturally with users, automate desktop tasks, fetch information, and respond intelligently using AI. It listens for a wake word, processes voice commands, and performs both online and offline actions.

<h2>Features</h2>
<br>

<b>Core Interaction</b>
<br>
- Wake-word activation (Nova)
<br>
- Voice recognition using microphone
<br>
- Text-to-speech responses
<br>
- Greeting system based on time of day
<br>
- Repeat last response memory
<br><br>

<b>Information & Intelligence</b>
<br>
- AI-powered conversational fallback using OpenAI
<br>
- Wikipedia quick search with spoken summaries
<br>
- Current time and date on voice command
<br>
- Latest news headlines via Google News RSS
<br>
- Joke generation
<br><br>

<b>Automation & Productivity</b>
<br>
- Open websites (Google, YouTube, Instagram, LinkedIn)
<br>
- Open desktop applications (Notepad, Calculator, Command Prompt, VS Code)
<br>
- Play music using custom music library
<br>
- Take and save screenshots
<br>
- System control (shutdown, restart, exit)
<br><br>
